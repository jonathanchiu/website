Fetchlr

In progress

A web application that fetches Tumblr posts based on a given username, and the specified post type. For example, giving 'tmagazine' as the username and selecting 'Text' as the post type will return all text posts posted by 'tmagazine.' Utilizes the Tumblr API for data retrieval and dynamic pagination for easy and organized navigation of the returned posts.

Still need to make more post types functional and design needs a lot of work. Basic functionality is there and currently only works with text, photo, and audio post types.

WP Apps

Complete

Implemented a number base converter that converts between binary, decimal, octal, and hexadecimal numbers, as well as a tip calculator, for the Windows Phone Store. Their original purpose was to learn/explore more about programming apps for the Windows Phone.

Time

In progress

Calculates time dilation experienced by a hypothetical observer from the perspective of an observer at "rest", given a duration of travel and a speed at which the moving observer desires to travel at. Will determine how much time has passed for our hypothetical moving observer relative to the observer at rest utilizing the lorentz factor.

Time dilation is really cool and incredibly bizarre.
